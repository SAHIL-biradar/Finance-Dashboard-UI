# Finance-Dashboard-UI
**💰Finance Dashboard UI**

A modern and responsive Finance Dashboard UI to track income, expenses, and financial data in a clean and user-friendly interface.

**🚀 Tech Stack**

Frontend: React, TypeScript

Styling: Tailwind CSS

UI Components: ShadCN UI

Charts: Recharts

Backend: Next.js API Routes

Database: PostgreSQL (Neon DB)

ORM: Drizzle ORM

Authentication: Clerk

**✨ Features**

📊 Dashboard with financial overview

💳 Track income and expenses

📁 Category-based transactions

📈 Interactive charts

🔐 Secure authentication

⚡ Fast and responsive UI

**⚙️ Setup**

1.Clone the repository:

git clone https://github.com/SAHIL-biradar/Finance-Dashboard-UI.git

cd Finance-Dashboard-UI

2.Install dependencies:

npm install

3.Run the project:

npm run dev


**📂 Project Structure**

app/ – Pages and routes

components/ – UI components

db/ – Database configuration

features/ – Core modules (transactions, accounts, etc.)
